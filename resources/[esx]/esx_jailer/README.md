# esx_jailer

made by Jager Bom


/!\ Purt the script under es_extended /!\

Only cops can put thiefs in jail with some commands:


/unjail id          
-->unjail someone


/jail1 id sec       
-->Put a person in jail 1


/jail2 id sec       
-->Put a person in jail 2


/jail3 id sec       
-->Put a person in jail 3
